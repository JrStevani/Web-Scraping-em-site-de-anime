Captura e Manipulação de Dados de Animes

Este projeto visa a captura de dados detalhados sobre animes e o armazenamento organizado desses dados em arquivos JSON.

Etapas do Projeto

1. Captura de URLs

Objetivo: O objetivo deste projeto é capturar URLs de animes dos gêneros "dublado" e "legendado" a partir do site animesonlinecc.to, organizando e armazenando essas URLs em arquivos JSON para facilitar futuras consultas ou análises.

Descrição: O script automatiza a coleta de URLs de animes por meio do Playwright, um framework para automação de navegadores. Ele navega por todas as páginas de cada gênero especificado, extrai as URLs dos animes, e salva esses links em arquivos JSON organizados em pastas específicas para cada gênero.

Tecnologias:

  • Python: Linguagem principal utilizada para o desenvolvimento do script.
  • Playwright: Biblioteca para automação de navegadores, usada para navegar e interagir com o site.
  • JSON: Formato utilizado para armazenar os dados de URLs coletados.
  • OS: Biblioteca para manipulação de diretórios e arquivos no sistema operacional.

Resultados: O script gera pastas separadas para os gêneros "dublado" e "legendado", contendo arquivos JSON com todas as URLs dos animes disponíveis em cada gênero. Esses arquivos podem ser usados para análises posteriores ou outras aplicações relacionadas a animes.

2. Extração de Informações

Objetivo: Coletar informações detalhadas de cada anime a partir das URLs capturadas.

Descrição: Para cada URL, extraímos informações como nome, descrição, gêneros, temporadas, episódios e imagem do anime.

Tecnologias: Playwright, Python.

Resultados: Dados extraídos e estruturados em um formato JSON.

3. Armazenamento dos Dados

Objetivo: O projeto visa garantir que todas as temporadas e episódios dos animes sejam capturados corretamente, identificando possíveis erros e registrando o progresso da captura.

Descrição: Para cada episódio, o Playwright navega até a página correspondente, extraindo o link do vídeo dentro de um iframe. Se o link for capturado com sucesso, ele é armazenado no arquivo JSON do respectivo anime. Caso contrário, erros são registrados para análise posterior.

Tecnologias: Python (manipulação de arquivos).

Resultados: Dados armazenados em um formato facilmente acessível e reutilizável.

Estrutura dos Arquivos

Pasta `animes`: Contém subpastas para cada anime, com arquivos JSON contendo as informações detalhadas.

Arquivos `Etapa 1.py`, `Etapa 2.py` e `Etapa 3.py`: Conjunto de Scripts realiza todo o processo de captura e armazenamento de dados.

Como Usar

  1. Clone este repositório.
  2. Instale as dependências usando pip install playwright beautifulsoup4 unidecode.
  3. Instale também playwright install.
  4. Execute os scripts `Etapa 1.py`, `Etapa 2.py` e `Etapa 3.py` para iniciar o processo de captura e armazenamento dos dados.
